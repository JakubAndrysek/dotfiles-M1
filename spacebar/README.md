# [Spacebar](https://github.com/cmacrae/spacebar)

## Installation

### Install fonts
- `brew tap homebrew/cask-fonts`
- `brew install --cask font-fontawesome`
- set `spacebar -m config icon_font` as `"Font Awesome 6 Free:Solid:12.0"`
